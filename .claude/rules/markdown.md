# Mermaid Diagrams

- Use vertical (top-to-bottom) layout
- Write all labels in English using ASCII characters only
- Use `<br>` within node labels for multi-line text where it improves readability
- Prefer tall, narrow diagrams over wide ones
- Never use `\n` for line breaks; use `<br>` instead
