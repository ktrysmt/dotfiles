# Chrome Extension Configs

## Overview
Browser extension configurations for Chrome/Chromium. Vimium provides keyboard-driven navigation with custom exclusion rules for web apps. uBlock Origin manages ad blocking and site filtering. Stylus injects custom CSS. uBlacklist filters search results. SiteBlock manages site access restrictions.

## Key Facts
- Vimium: exclusions for Gmail, Outlook, Twitter, Drive, Notion, Slack, GitHub Dev, ClickUp, Atlassian, Asciiflow; custom maps: gr/gl/go (close tabs), gf (link hints), gp (pin tab)
- uBlock Origin: whitelist/blacklist domain rules
- Stylus: custom CSS injection for websites
- uBlacklist: search result filtering rules
- SiteBlock: site access restriction list
